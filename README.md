# Puppet_of_Hourai-muze
Seven-probe memory ranker for local AI companion.

## 技术栈
Python + SQLite + FAISS + Doubao Embedding + DeepSeek API

## 核心功能
1.  七探针记忆卡片重拾逻辑：
    - 关键词分
    - 语义分
    - 重要性分
    - 锚定分
    - 生长分
    - 时间爆发
    - 衰减惩罚
2.  国内可落地：用豆包Embedding（2048维）替代谷歌Vetex，DeepSeek VA唤醒度双轨并行，缓解token爆炸与上下文遗忘问题
3.  本地轻量化部署，数据独立存储
